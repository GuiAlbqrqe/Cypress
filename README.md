# ✅ Automação E2E com Cypress

Projeto de automação de testes End-to-End (E2E) usando **Cypress**, com foco em validar fluxos críticos da aplicação.

![Cypress](https://img.shields.io/badge/tested%20with-Cypress-04C38E.svg)

---

## 🧰 Tecnologias

- Cypress
- JavaScript
- Node.js
- Git/GitHub

---

## 🎯 Objetivo

Automatizar cenários de teste para garantir que funcionalidades principais estejam funcionando, como:

- Login
- Navegação em páginas
- Cadastro e validação de formulários
- Fluxos completos de ponta a ponta (E2E)
